# SIE Prep

A free, open-source study platform for the FINRA Securities Industry
Essentials (SIE) exam. Built with React, Vite, TypeScript, and Tailwind CSS —
no backend, all content in structured JSON, progress saved in localStorage.

> **Disclaimer:** This project is for study purposes only and is not
> affiliated with, endorsed by, or sponsored by FINRA.

## Features (planned)

- Chapter-based reading guides mapped to FINRA's official content outline
- End-of-chapter unit tests
- Weighted 75-question timed practice exams
- Spaced-repetition flashcards
- Missed-question review bank
- Progress dashboard with test-readiness tracking

## Local setup

```bash
npm install
npm run dev
```
